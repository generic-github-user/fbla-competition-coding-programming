1. Go to the admin Dashboard
  2. Click on a student's name - optionally, search for a student using the magnifying class in the upper left
  3. This will open a dialog box with the Student Overview
  4. Edit any of the fields you want:
    1. Student name
    2. Student number
    3. Grade
    4. Total service hours
  5. Click "Update" at the bottom of the overview box, and then "Close" to close the panel
