1. Go to the admin Dashboard
  2. Click on the name of a student to view the overview window for that student
  3. Click "View Report" to see more information about that student
  4. Click the "Add Volunteer Hours" button in the lower right-hand corner of the screen
  5. Fill out the fields of the form that appears:
    1. Service event description
    2. Hours
    3. Date
  6. Click "Add"
